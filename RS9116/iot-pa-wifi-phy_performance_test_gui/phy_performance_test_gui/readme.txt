------------------------------------------------------------  README  -------------------------------------------------------------

This project helps the user to analyze the performance of RS9116 NCP module.

-----------------------------------------------------------------------------------------------------------------------------------

Note: 
1. Please refer the readme.md for execution of the application.
2. Executable applications are available for Windows and macOS.
3. The executable applications require no installation of python and its packages.
4. The python script can be executed on Windows, macOS and Linux after installing python and its packages.

----------------------------------------------------------------------------------------------------------------------------------

Files:
1.	\applications
		   This folder contains executable applications for Windows and macOS.

2.	\code_base
		   This folder contains the python script and the associated images for running the PHY Performance Test GUI application.

3.	\python_installation
		   This folder contains files for installing python and its packages.

4.	\resources
		   This folder contains the images associated with readme.md file.
		   
5.	\README.md
		   This is the user guide for PHY Performance Test GUI
----------------------------------------------------------------------------------------------------------------------------------
