
EFR32 Pin Configurations:
-------------------------
1.Connect  RS9116 SPI header J4_PIN-2 to EFR32 CS #0 PC03(PIN10)   for CS
2.Connect  RS9116 SPI header J4_PIN-3 to EFR32 GND PIN1 for GND
3.Connect  RS9116 SPI header J4_PIN-4 to EFR32 3V3 PIN20 for VCC
4.Connect  RS9116 SPI header J4_PIN-5 to EFR32 SPI_CLK #0 (PC02) (PIN10)for clock
5.Connect  RS9116 SPI header J4_PIN-7 to EFR32 MOSI (PC00)(PIN6) for MOSI
6.Connect  RS9116 SPI header J4_PIN-8 to EFR32 MISO (PC01)(PIN6) for MISO
7.Connect  RS9116 SPI header J4_PIN-9 to EFR32 INTR (PB01)(PIN13) for INTERRUPT
